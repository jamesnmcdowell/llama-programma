# Llama Programma

> Llama Programma is a fullstack JavaScript ecommerce web app built with react, redux, css grid/flexbox, and node.

## Setup
```shell
$ git clone git@github.com:jamesnmcdowell/llama-programma.git
$ yarn install 
$ yarn start
```
